Bot Or Not

A web based application where muliple people can log on and play. The object of the game is to chat with multiple people and a chat bot. At the end of 30 seconds, you have the opportunity to guess which person is the bot. The player and bots name are changed to random names to avoid cheating.

Bot or Not was a team project consisting of 3 members. This web app was built using Java, Javascript, JSP, CSS, MySQL, Spring Boot, CleverBot API, and Web Sockets.

My role was helping with the backend and some front-end concept.
